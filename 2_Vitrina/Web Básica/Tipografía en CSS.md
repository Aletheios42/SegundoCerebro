```css
.texto {

    font-weight: bold;
    line-height: 1.5;
    text-align: center;
    text-transform: uppercase;
}
```
### **Unidades de tamaño**
- `px` → Tamaño absoluto.
- `em` → Relativo al tamaño del elemento padre.
- `rem` → Relativo al `font-size` del `html`.
- `%` → Proporcional al contenedor.
